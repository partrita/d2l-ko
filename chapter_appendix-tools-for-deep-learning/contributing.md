# 기여하기 (Contributing to Dive into Deep Learning)
:label:`sec_contributing`

이 책은 커뮤니티의 기여를 통해 성장하는 오픈 소스 프로젝트입니다.
오타 수정부터 새로운 섹션 작성까지, 모든 기여를 환영합니다.
기여하는 방법에 대한 자세한 가이드는 프로젝트의 [기여 가이드](https://github.com/d2l-ai/d2l-en/blob/master/CONTRIBUTING.md)를 참조하십시오.

## 기여 절차 (General Workflow)

1. GitHub에서 저장소를 포크(fork)합니다.
2. 로컬 머신에 포크한 저장소를 클론(clone)합니다.
3. 새로운 브랜치를 생성하여 작업을 진행합니다.
4. 변경 사항을 커밋(commit)하고 포크한 저장소에 푸시(push)합니다.
5. 원본 저장소로 풀 리퀘스트(Pull Request)를 보냅니다.

## 유용한 팁 (Tips)

* 오타나 간단한 수정은 GitHub 웹 인터페이스에서 직접 수행할 수 있습니다.
* 큰 변경 사항의 경우, 작업을 시작하기 전에 이슈(issue)를 열어 논의하는 것이 좋습니다.
* 스타일 가이드와 서식 규칙을 준수해 주십시오.

여러분의 소중한 기여가 이 책을 더 좋게 만듭니다!
