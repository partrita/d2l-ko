# Google Colab 사용하기 (Using Google Colab)
:label:`sec_colab`

우리는 :numref:`sec_sagemaker`와 :numref:`sec_aws`에서 AWS에서 이 책을 실행하는 방법을 소개했습니다. 다른 옵션은 Google 계정이 있는 경우 [Google Colab](https://colab.research.google.com/)에서 이 책을 실행하는 것입니다.

Colab에서 섹션의 코드를 실행하려면 :numref:`fig_colab`에 표시된 것처럼 `Colab` 버튼을 클릭하기만 하면 됩니다.

![Colab에서 섹션의 코드를 실행하십시오.](../img/colab.png)
:width:`300px`
:label:`fig_colab`


코드 셀을 처음 실행하는 경우, 
:numref:`fig_colab2`와 같은 경고 메시지를 받게 됩니다. 
무시하려면 "RUN ANYWAY"를 클릭하십시오.

![무시하려면 "RUN ANYWAY"를 클릭하십시오.](../img/colab-2.png)
:width:`300px`
:label:`fig_colab2`

다음으로, Colab은 이 섹션의 코드를 실행하기 위해 인스턴스에 연결해 줄 것입니다. 
특히 GPU가 필요한 경우, 
GPU 인스턴스에 연결하기 위해 Colab이 자동으로 요청될 것입니다.


## 요약 (Summary)

* Google Colab을 사용하여 이 책의 각 섹션 코드를 실행할 수 있습니다.
* 이 책의 모든 섹션에서 GPU가 필요한 경우 GPU 인스턴스에 연결하기 위해 Colab이 요청될 것입니다.


## 연습 문제 (Exercises)

1. Google Colab을 사용하여 이 책의 임의의 섹션을 여십시오.
2. Google Colab을 사용하여 GPU가 필요한 임의의 섹션을 편집하고 실행하십시오.
